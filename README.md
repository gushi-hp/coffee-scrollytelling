# 一颗咖啡豆的旅程 · The Journey of a Bean

一个杂志摩登风格的滚动叙事网页，讲述一颗咖啡豆从产地到杯中的 7 章故事。

## 在线预览

https://gushi-hp.github.io/coffee-scrollytelling/

## 操作

- 滚轮 / 方向键 / 空格 / PageUp/PageDown：翻页
- 触屏：上下滑动
- 右侧圆点：直接跳转章节

## 技术

- 单文件 HTML（CSS/JS 内联）
- GSAP 3.12.5（章节转场动画）
- Canvas 粒子系统（研磨页咖啡粉飘动效果）
- CSS backdrop-filter 毛玻璃 / Google Fonts 杂志字体
